10/07 오늘의 배운 것
  • input은 form 안에 넣어야 추후 submit을 작동시킬 수 있다.
  • input 안에 같이 쓰는 Teg는
    • id
    • type
    • name
    • value
    • size
    • maxlength 등 외에 더 있음
  • input 사용 시 주로 같이 쓰는 type은
    • text
    • password
    • file : 파일 첨부
    • reset : 입력 정보를 초기화
    • submit : 어딘가에 입력한 정보를 전송(주로 서버) (image로 일부 대체 가능)
    • checkbox : 중복선택 가능
    • radio : name이 같을 경우 중복선택 불가
      (checkbox와 radio는 checked 사용 시 해당 input 선택을 디폴트로 한다.)
  • input 부가 설정 Teg
    • placeholder : input창에 회색글씨 아웃
    • autofocus : 페이지 열었을때 자동으로 커서 지정되는 곳
    • required : 사용자가 필수로 입력해야하는 부분
    • readonly : 읽기전용(사용자가 입력불가)
    • autocomplete : 자동완성기능 설정(이전에 입력했던 정보들을 뜨게할건지 안뜨게 할건지)
